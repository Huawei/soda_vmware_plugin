Unit tests Requirement for VRO 
================================
1. Before running unit tests, make sure you have a working OpenSDS Hotpot environment.
2. Set required fields in //test//resources//config.properties
3. To set volume Id, create a volume in OpenSDS manually and give the volume ID in  config.properties
4. Set eclipse project to run unit tests and coverage.
5. Install Junit5 (Java build path -> Librararies -> Add library -> Junit5
